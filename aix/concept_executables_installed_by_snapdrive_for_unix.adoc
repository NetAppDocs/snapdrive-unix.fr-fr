= Exécutables installés par SnapDrive pour UNIX
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive pour UNIX installe des exécutables pour le système d'exploitation AIX.

Ces exécutables sont installés à `/opt/NetApp/snapdrive/bin/snapdrive`.
